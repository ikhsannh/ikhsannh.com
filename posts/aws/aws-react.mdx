## Launch EC2 Instance in AWS from template

Menggunakan template yang telah dibuat sebelumnya.
![TemplateInstance](https://github.com/ikhsannh/ikhsannh.com/blob/main/public/aws/aws-react/TemplateInstance.jpeg)

Menambahkan Key-Pair baru saat peluncuran instance.
![KeyPair](https://github.com/ikhsannh/ikhsannh.com/blob/main/public/aws/aws-react/NewPair.jpeg)

Meluncurkan instance.
![LaunchInstance](https://github.com/ikhsannh/ikhsannh.com/blob/main/public/aws/aws-react/LaunchInstance.jpeg)

## Konfigurasi Proyek React untuk AWS

Membuat ekosistem PM2 untuk proyek React.
Mengatur kunci-pasangan yang sudah diluncurkan.

## Git Commit dan Push ke Private Repo GitHub

Mengcommit dan mendorong perubahan proyek React ke repo GitHub pribadi.

## Koneksi ke Instance AWS melalui SSH

Menghubungkan ke instance AWS menggunakan SSH.
Membuka terminal Ubuntu pada instance.
Menjadi pengguna root dengan perintah "sudo su".
Memperbarui paket Ubuntu dengan "sudo apt update".

## Persiapan Lingkungan AWS Ubuntu

Memeriksa apakah Node.js sudah terinstal, jika tidak, menginstal NVM.
Memeriksa apakah Git sudah terinstal, jika tidak, menginstal Git.
Mengatasi masalah izin SSH dengan menghasilkan dan menambahkan SSH key.
Mengklon repo GitHub proyek React ke dalam instance.

## Konfigurasi Keamanan

Mengatur izin akses kunci-pasangan agar hanya dapat diakses oleh pemilik dengan "chmod 400 key.pem".

## Instalasi Nginx dan PM2

Menginstal Nginx sebagai server web.
Menginstal PM2 untuk mengelola aplikasi React.

## Konfigurasi Nginx untuk Hosting Proyek

Mengonfigurasi Nginx agar dapat meng-host proyek React di server.
Melakukan restart Nginx untuk menerapkan konfigurasi.

## Pengecekan di Domain

Memeriksa proyek React yang di-hosting di domain yang sesuai.
